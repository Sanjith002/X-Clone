🐦 X (Twitter) Clone – MERN Stack

A full-stack social media application inspired by X (Twitter), built using the MERN stack. It allows users to create posts, interact with content, and manage their profiles.

---

🌐 Live Demo

👉 https://x-clone02.pages.dev/

---

🚀 Features

- User authentication (login & signup)
- Create, edit, and delete posts
- Like and unlike posts
- Dynamic feed
- User profile management

---

🛠️ Tech Stack

- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB

---

📌 Highlights

- RESTful API integration
- Responsive UI with Tailwind CSS
- Scalable full-stack architecture

---

📈 Future Improvements

- Comments & replies
- Notifications
- Media upload (images/videos)
